# GuardianHub：面向高校场景的 AI 数字安全防护平台接口文档

后端默认地址：`http://127.0.0.1:8000`

所有模块统一使用 `high`、`medium`、`low` 风险等级。

## GET /api/health

检查后端服务是否正常运行。

```json
{
  "status": "ok",
  "message": "GuardianHub backend is running"
}
```

## POST /api/detect

Privacy Sentinel 图片隐私检测接口。接收用户上传的图片，保存到 `static/uploads/`，并返回 mock 隐私检测结果。

请求类型：`multipart/form-data`

字段：

- `file`：图片文件

响应示例：

```json
{
  "imageId": "img_xxx",
  "originalImageUrl": "/static/uploads/img_xxx.png",
  "riskLevel": "high",
  "summary": "检测到手机号、地址、二维码等高风险隐私信息，建议打码后再分享。",
  "items": [
    {
      "id": "img_xxx_phone",
      "type": "phone",
      "label": "手机号",
      "text": "138****1234",
      "riskLevel": "high",
      "box": { "x": 120, "y": 250, "width": 260, "height": 50 },
      "suggestion": "手机号属于高风险隐私信息，建议遮盖后再分享。"
    }
  ]
}
```

## POST /api/mask

根据 `imageId`、`maskType` 和待处理区域对图片进行打码处理，处理后的图片保存到 `static/processed/`。

```json
{
  "imageId": "img_xxx",
  "maskType": "black",
  "items": [
    { "x": 120, "y": 250, "width": 260, "height": 50 }
  ]
}
```

`maskType` 支持：

- `black`：黑条遮盖
- `blur`：模糊处理
- `mosaic`：马赛克

响应示例：

```json
{
  "processedImageUrl": "/static/processed/img_xxx_safe.png",
  "message": "已完成安全处理。"
}
```

## GET /api/history

返回最近图片隐私检测记录。

## POST /api/scam/analyze

Scam Radar 反诈雷达接口。输入聊天文本，基于规则返回诈骗风险等级、风险原因和建议操作。

请求示例：

```json
{
  "text": "奖学金补贴今日截止，请点击链接填写银行卡和验证码。"
}
```

响应示例：

```json
{
  "riskLevel": "high",
  "score": 100,
  "reasons": [
    {
      "label": "索要验证码",
      "evidence": "填写银行卡和验证码",
      "riskLevel": "high"
    }
  ],
  "suggestions": [
    "停止转账、提交验证码或填写银行卡。",
    "通过学校官网、辅导员或官方客服电话二次确认。"
  ]
}
```

## POST /api/link/check

Link Guard 链接卫士接口。输入 URL，检查 HTTPS、短链接、可疑关键词和异常域名。

请求示例：

```json
{
  "url": "http://bit.ly/scholarship-verify"
}
```

响应示例：

```json
{
  "riskLevel": "high",
  "normalizedUrl": "http://bit.ly/scholarship-verify",
  "checks": [
    {
      "label": "未使用 HTTPS",
      "evidence": "http",
      "riskLevel": "medium"
    }
  ],
  "suggestions": [
    "不要直接打开该链接。",
    "使用学校或服务商官网入口重新访问。"
  ]
}
```

## POST /api/doc/check

Doc Shield 提交护盾接口。当前为 mock 检查，展示文件命名、隐私字段和材料清单检查结果。

请求示例：

```json
{
  "fileName": "张三_计算机学院_创新创业报名表.docx"
}
```

响应示例：

```json
{
  "riskLevel": "medium",
  "checks": [
    {
      "label": "隐私字段待复核",
      "evidence": "检测到姓名、学院等字段，提交前建议确认是否符合材料要求。",
      "riskLevel": "medium"
    }
  ],
  "checklist": [
    { "item": "文件命名检查", "status": "pass" },
    { "item": "隐私检查", "status": "warning" },
    { "item": "材料清单检查", "status": "pending" }
  ],
  "suggestions": [
    "提交前确认文件名包含姓名、学院、材料类型等关键信息。",
    "删除无关手机号、身份证号、家庭住址等隐私字段。"
  ]
}
```
