# GuardianHub 接口文档

后端默认地址：`http://127.0.0.1:8000`

## 通用约定

所有风险结果统一使用：

- `riskLevel`: `high` / `medium` / `low`
- `score`: 0-100，分数越高越安全
- `summary`: 检测摘要
- `suggestions`: 建议操作列表

## 健康检查

`GET /api/health`

```json
{
  "status": "ok",
  "message": "GuardianHub backend is running"
}
```

## Privacy Sentinel 隐私哨兵

### 图片检测

`POST /api/detect`

表单字段：

- `file`: 图片文件

返回：

- `imageId`
- `originalImageUrl`
- `riskLevel`
- `summary`
- `items`: 隐私检测项列表，包含 `box`、`label`、`riskLevel`、`suggestion`

### 图片打码

`POST /api/mask`

```json
{
  "imageId": "img_xxx",
  "maskType": "black",
  "items": [{ "x": 10, "y": 10, "width": 100, "height": 40 }]
}
```

`maskType` 支持 `black`、`blur`、`mosaic`。

## Code Guardian 代码卫士

`POST /api/code/analyze`

JSON 请求：

```json
{
  "language": "python",
  "code": "api_key = \"sk-demo\""
}
```

也支持 `multipart/form-data`：

- `language`
- `file`

支持文件后缀：`.py`、`.java`、`.js`、`.ts`、`.sql`、`.txt`。当前不支持 zip 项目级扫描。

返回：

- `riskLevel`
- `score`
- `summary`
- `language`
- `vulnerabilities`: 风险项列表，包含 `id`、`type`、`title`、`riskLevel`、`line`、`snippet`、`reason`、`suggestion`
- `suggestions`
- `shouldSubmit`

## Link Guard 链接卫士

`POST /api/link/check`

```json
{
  "url": "https://example.com/login?redirect=pay&token=xxx",
  "source": "短信"
}
```

返回：

- `riskLevel`
- `score`
- `summary`
- `normalizedUrl`
- `checks`: 链接体检项目，覆盖 HTTPS、短链接、IP 直连、域名长度、URL 长度、关键词、参数、随机 token、仿冒域名
- `suspiciousParams`
- `sourceRisk`
- `suggestions`
- `shouldOpen`

## Doc Shield 提交护盾

`POST /api/doc/check`

表单字段：

- `requirement_text`: 提交要求文本
- `files`: 一个或多个材料文件

返回：

- `riskLevel`
- `score`
- `summary`
- `parsedRequirements`: 解析出的格式、命名规则、必需材料、字数页数、截止时间
- `files`: 上传材料摘要
- `checks`: 材料完整性、格式规范、隐私风险检查项
- `suggestions`

## 历史记录

`GET /api/history`

返回最近图片隐私检测与处理记录。前端安全中心会将后端历史和当前会话中的 Code Guardian、Link Guard、Doc Shield 报告合并展示为最近安全检测历史。
