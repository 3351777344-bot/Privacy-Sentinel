# GuardianHub 接口文档

后端默认地址：`http://127.0.0.1:8000`

## 通用约定

所有风险结果统一使用：

- `riskLevel`: `high` / `medium` / `low`
- `score`: 0-100，分数越高越安全
- `suggestions`: 建议操作列表
- `TextFinding`: `{ label, evidence, riskLevel }`

## 健康检查

`GET /api/health`

返回：

```json
{
  "status": "ok",
  "message": "GuardianHub backend is running"
}
```

## 隐私哨兵

### 图片检测

`POST /api/detect`

表单字段：

- `file`: 图片文件

返回：

- `imageId`
- `originalImageUrl`
- `riskLevel`
- `summary`
- `items`: 隐私检测项列表，包含 `box`、`label`、`riskLevel`、`suggestion`

### 图片打码

`POST /api/mask`

请求体：

```json
{
  "imageId": "img_xxx",
  "maskType": "black",
  "items": [{ "x": 10, "y": 10, "width": 100, "height": 40 }]
}
```

`maskType` 支持 `black`、`blur`、`mosaic`。

## 反诈雷达

`POST /api/scam/analyze`

请求体：

```json
{
  "text": "聊天记录文本"
}
```

返回：

- `riskLevel`
- `score`
- `reasons`: 风险证据，覆盖诱导转账、高额回报、催促决策、脱离平台、隐瞒他人、可疑链接等类型
- `suggestions`

## 链接卫士

`POST /api/link/check`

请求体：

```json
{
  "url": "https://example.com"
}
```

返回：

- `riskLevel`
- `normalizedUrl`
- `checks`: 链接体检项，覆盖 HTTPS、短链接、可疑关键词、异常域名、URL 过长、随机字符、可疑参数
- `suggestions`

## 提交护盾

`POST /api/doc/check`

表单字段：

- `requirement_text`: 提交要求文本
- `files`: 一个或多个材料文件

返回：

- `riskLevel`
- `score`
- `summary`
- `parsedRequirements`: 解析出的格式、命名规则、必需材料、字数页数、截止时间
- `files`: 上传材料摘要
- `checks`: 材料完整性、格式规范、隐私风险检查项
- `suggestions`

## 历史记录

`GET /api/history`

返回最近图片隐私检测与处理记录。前端安全中心会将后端历史和当前会话中的反诈、链接、提交护盾报告合并展示为最近安全检测历史。
