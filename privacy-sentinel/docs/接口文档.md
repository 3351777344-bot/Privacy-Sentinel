# Privacy Sentinel 接口文档

后端默认地址：`http://127.0.0.1:8000`

## GET /api/health

用于检查后端服务是否正常运行。

响应示例：

```json
{
  "status": "ok",
  "message": "Privacy Sentinel backend is running"
}
```

## POST /api/detect

接收用户上传的图片，保存到 `static/uploads/`，并返回 mock 隐私检测结果。

请求类型：`multipart/form-data`

字段：

- `file`：图片文件

响应示例：

```json
{
  "imageId": "img_xxx",
  "originalImageUrl": "/static/uploads/img_xxx.png",
  "riskLevel": "high",
  "summary": "检测到手机号、地址、二维码等高风险隐私信息，建议打码后再分享。",
  "items": [
    {
      "id": "img_xxx_phone",
      "type": "phone",
      "label": "手机号",
      "text": "138****1234",
      "riskLevel": "high",
      "box": { "x": 120, "y": 250, "width": 260, "height": 50 },
      "suggestion": "手机号属于高风险隐私，建议隐藏后再分享。"
    }
  ]
}
```

## POST /api/mask

根据 `imageId`、`maskType` 和待处理区域对图片进行真实打码处理，处理后的图片保存到 `static/processed/`。

请求示例：

```json
{
  "imageId": "img_xxx",
  "maskType": "black",
  "items": [
    { "x": 120, "y": 250, "width": 260, "height": 50 }
  ]
}
```

`maskType` 支持：

- `black`：黑条遮挡
- `blur`：高斯模糊
- `mosaic`：马赛克

响应示例：

```json
{
  "processedImageUrl": "/static/processed/img_xxx_safe.png",
  "message": "已完成安全处理。"
}
```

## GET /api/history

返回最近检测记录。

响应字段：

- `imageId`
- `originalImageUrl`
- `processedImageUrl`
- `riskLevel`
- `summary`
- `createdAt`
- `status`
