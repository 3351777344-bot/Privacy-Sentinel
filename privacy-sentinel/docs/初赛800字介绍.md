# 初赛 800 字介绍

Privacy Sentinel（隐私哨兵）是一款面向图片分享场景的 AI 隐私安全分享助手。用户在社交平台、校园群、售后沟通或办公协作中，经常需要分享截图、聊天记录、快递单、订单页面、证件照片等图片。这些图片里往往包含手机号、收货地址、二维码、订单号、头像昵称、学号等隐私信息。传统做法依赖用户自己发现风险，再使用图片编辑工具手动涂抹，流程繁琐且容易遗漏。

本项目希望把隐私保护前置到“分享之前”。当用户上传图片后，系统会自动生成隐私检测结果，在图片上标注敏感区域，并按照高风险、中风险、低风险进行分级。对于手机号、地址、二维码等高风险信息，系统会给出明确处理建议，并支持一键打码。当前 Demo 已实现黑条、模糊、马赛克三种真实图像处理方式，用户可以在安全预览中确认处理效果，再决定是否分享。

当前阶段项目采用 PC Web Demo 进行初赛验证：前端使用 React + Vite + TypeScript，后端使用 FastAPI，图像处理使用 Pillow，检测结果先通过 mock 数据生成，保证演示稳定可靠。后续将逐步接入 OCR、二维码识别、人脸检测和更完善的风险分类能力，使检测结果从原型验证升级为真实智能识别。

Privacy Sentinel 的创新点在于，它不是普通图片编辑器，而是围绕“分享前隐私风险检测”构建完整闭环：主动检测、风险分级、区域标注、一键处理、安全预览和历史记录。复赛阶段，项目计划迁移到 HarmonyOS ArkTS 前端，进一步发挥 HarmonyOS 多端协同能力：手机端用于快速检测和分享，平板端用于大屏精修与批量处理，手表端用于高风险提醒，最终形成面向日常分享场景的轻量化 AI 隐私防护工具。
