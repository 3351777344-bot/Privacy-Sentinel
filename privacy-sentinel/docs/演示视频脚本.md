# Privacy Sentinel 演示视频脚本

目标时长：5 分钟以内

## 0:00-0:40 项目背景

大家好，我们的项目是 Privacy Sentinel，中文名“隐私哨兵”。它是一款 AI 隐私安全分享助手，解决用户在分享截图、聊天记录、快递单、订单页面时无意泄露手机号、地址、二维码、订单号等隐私信息的问题。

## 0:40-1:20 核心思路

项目的核心不是做普通图片编辑器，而是把隐私保护前置到分享之前。用户上传图片后，系统主动检测隐私风险，标注敏感区域，给出风险分级，并支持一键生成安全分享版图片。

## 1:20-2:50 功能演示

打开 PC Web Demo。首先上传一张示例图片，点击“开始隐私检测”。系统返回检测结果，在图片上标出手机号、地址、二维码、订单号和昵称等区域。右侧风险概览展示整体风险等级、检测项数量和高/中/低风险统计。下方列表展示每个隐私项的识别文本和处理建议。

接着进入一键打码区域，选择黑条、模糊或马赛克。点击“一键处理高风险信息”，后端会根据检测框对原图进行真实处理。完成后，安全预览区域展示处理后的图片，并给出分享建议。历史记录会同步更新检测和处理状态。

## 2:50-3:40 技术说明

当前阶段是初赛验证版，采用 React + Vite + TypeScript 构建前端，FastAPI 提供接口，Pillow 完成真实图像打码。检测结果暂时使用 mock 数据，避免过早引入复杂 OCR 或付费大模型 API，优先保证本地 Demo 稳定运行。

## 3:40-4:30 创新点

Privacy Sentinel 的创新点在于形成了“主动检测、风险分级、区域标注、一键处理、安全预览”的完整闭环。它关注的是分享前隐私风险，而不是泛化的图片编辑。

## 4:30-5:00 总结与规划

后续我们会接入 OCR、二维码识别、人脸检测和更完善的风险分类模型，并迁移到 HarmonyOS ArkTS 前端。手机端用于快速检测和分享，平板端用于大屏精修与批量处理，手表端用于高风险提醒，形成 HarmonyOS 多端协同的 AI 隐私安全分享体验。
